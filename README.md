# Introduction

Data analysis in transport has become increasingly important in the era of big data. This book introduces the fundamentals of data analysis using R and explores its applications in the transport field. Through data analysis and data visualisation, readers learn how to organise and process data effectively while gaining a clearer understanding of data analysis strategies.
